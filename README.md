Extension for Firefox, Chrome, Opera, and Yandex Browser browsers.

Highlights problematic elements from the [Site Alarm](https://site-alarm.com/en/tools/page-checker) service report.

## How to use it? ##

1. Install it from your browser's app marketplace.
2. In the [Site Alarm](https://site-alarm.com/en/tools/page-checker) service report click on the xpath code of the element and click the "Show on page" button.
