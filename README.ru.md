Расширение для браузеров Firefox, Chrome, Opera, Yandex Browser.

Подсвечивает проблемные элементы из отчёта сервиса [Site Alarm](https://site-alarm.com/tools/page-checker).

## Как использовать? ##

1. Установить его из маркетплейса приложений вашего браузера.
2. В отчёте сервиса [Site Alarm](https://site-alarm.com/tools/page-checker) кликнуть по xpath коду элемента и нажать кнопку «Показать на странице».
